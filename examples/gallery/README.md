# Gallery

Small BoardLang programs that show one idea each.

| Program | Idea |
|---------|------|
| `counter.board` | Named registers with `let` / `inc` / `sayreg`. |
| `tags.board` | PGN roster tags (`[Section "..."]`) stored on the chess object for `{pgn}`. |

Open any `.board` file with the BoardLang syntax extension (see `editor/boardlang-syntax/README.md`).
